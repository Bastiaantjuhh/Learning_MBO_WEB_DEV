Mijn eerste webpagina

Dit is mijn eerste webpagina en daar ben ik trots op. Het maakt nog geen gebruik van css, maar dat komt later wel. Als iets van belang is (zoals het woord trots in de vorige zin) of iets is een Engelse term (zoals Hyper Text Markup Language) wordt in HTML5 een speciaal element (of tag) gebruikt. Zo heeft ieder element een eigen betekenis.

De opmaak van een webpagina, dus hoe iets er uit ziet wordt in CSS bepaald. Als niet specifiek is opgegeven hoe die opmaak is, dan gebruikt de browser een eigen standaard opmaak. Dit wordt de default styling genoemd.
